# 钢条切割

题目在《算法导论》书中的第204页。下面简单的描述一下题目。

有一根钢条，给出一个价格数组`price[] = {0, 1, 5, 8, 9, 10, 17, 17, 20, 24, 30}`，其中`price[1]`表示长度为`1`的钢条价格为`1`，`price[2]`表示长度为`2`的钢条价格为`5`，求最多能卖多少钱？
